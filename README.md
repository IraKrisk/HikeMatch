# HikeMatch

**Note:** Learning project from 2018. For current professional work, see pinned repositories.

Final year college project at National College of Ireland - a social meetup application for connecting hikers, built with Node.js.

## Tech Stack
- Node.js
- Express.js
- Handlebars

## Project Context
This was an educational project completed during my studies at National College of Ireland, focused on learning server-side JavaScript development and route matching algorithms.
